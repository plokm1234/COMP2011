Eclipse Overview

Eclipse in C++ provides an Integrated Development Environment (IDE) that allows developers to write, build, and debug C++ programs. Eclipse itself is written in Java and can be used in multiple platforms (Microsoft Windows, Linux and Mac OS X). Except C++, Eclipse can also be used to develop applications in various programming languages such as Java, PHP, Python and more.

Using Eclipse on a USB thumb drive

(Warnings: Installing Eclipse on a USB thumb drive is portable, but Eclipse usually runs slower on a USB thumb drive. If you usually bring your own laptop, it is better to install Eclipse on your own laptop)

Step 1: Plug a USB thumb drive to a computer

Record down the drive letter. It may be different from yours. In this example, the drive letter is H:\. About 500MB free space is required to install Eclipse on a USB thumb drive.

Step 2: Create an empty folder to store the Eclipse

Eclipse is very portable. You can simply install Eclipse in an USB thumb drive. The first step is to create an empty folder (e.g. H:\usb_eclipse).

Step 3: Extract the Eclipse to the empty folder

After you have unzipped the Eclipse to the empty folder (located at H:\usb_eclipse), you can delete the zip file.

Step 4: Create an empty folder to store all the C++ projects

In Eclipse, we need to specify a location to store all the C++ projects (i.e. all the C++ source code files, executable files and other system files). We call this location a workspace. In this example, we create an empty folder located at H:\usb_eclipse\workspace

Step 5: Start Eclipse on the USB thumb drive

Enter the folder (H:\usb_eclipse\eclipse3.5-cpp-win) and double-click eclipse.exe

Note: you may need to setup the MinGW and JAVA path before running the Eclipse.
See the sections "Eclipse (Microsoft Windows) (* Read it at home *)" or "Eclipse (Mac OS X) (* Read it at home *)"

Step 6: Specify the workspace directory (H:\usb_eclipse\workspace) and click "Ok"

Now, all the C++ source codes will be placed inside the workspace folder.

Developing a HelloWorld application using Eclipse

Step 1: Launch Eclipse

(For details, see "Using Eclipse/C++ in the labs" or "Using Eclipse on a USB thumb drive")

Step 2: Select the workspace

Workspace stores all your C++ source codes.
1. Please click "browse" to choose a path that you can memorize (e.g. Your desktop). In this example, a path "C:\Temp" is used.
2. After that, click "OK".

Step 3: Close the Welcome screen

Step 4: Create a new C++ project

Project helps you to organize your C++ source code file(s).

Step 5: Give the name of a project and select the C++ compiler tool

A compiler converts C++ source codes to object codes. These object codes can be used to create executable files

Step 6: Add a source code file to the project

Step 7: Name the source code file

You can choose your favorite file name. In this example, we use "main.cpp". Then click Finish.

Step 8: Edit and save the C++ source code file

** Notice ** remember to save the file. A star (*) is used to indicate that the file is unsaved. You will encounter some unknown errors if you forget to save the file.

Step 9: Click the hammer button to compile the source code

Step 10: Click the play button to execute the program and view the output result.

Eclipse (Microsoft Windows) (* Read it at home *)

Step 1: Download the Eclipse/C++ package for Mircosoft Windows

The package can be downloaded here (eclipse-4.4-cpp-win.zip)

You can download this file into any folder in your PC. See the figure below:

Step 2: Extract the file using WinZip or WinRar

Step 3: Setup the MinGW and JAVA path

Before you run the Eclipse, please set the Java and MinGW paths in your window environment. To know how to setup a window environment path: you can visit: http://www.computerhope.com/issues/ch000549.htm.

  1. Java Path - For example, if you have installed a JDK (Java Development Kit: http://www.oracle.com/technetwork/java/javase/downloads/index.html), the Java path is:
    C:\Progra~1\Java\jre7\bin (please check your java installation path)
  2. MinGW Paths - For example, if you unzip your Eclipse under C drive, the mingw paths are : C:\eclipse-4.4-cpp-win\mingw\bin and C:\eclipse-4.4-cpp-win\mingw\msys\1.0\bin
  3. You should not delete your original path. You should add a semi-colon to separate the new path. Then the path should be: C:\Progra~1\Java\jre7\bin;C:\eclipse-4.4-cpp-win\mingw\bin;C:\eclipse-4.4-cpp-win\mingw\msys\1.0\bin;%PATH%

Step 4: Launch Eclipse
