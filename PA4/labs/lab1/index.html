Test your Eclipse Setup

Simply copy the program below into your Eclipse editing window, and then compile and execute the program to ensure that your Eclipse programming environment is properly setup. 

#include <iostream>
using namespace std;

int main()
{
   int x, y, z;

   x = 1;
   y = 2;
   z = x+y;

   cout << "The sum of " << x << " and " << y << " is " << z << endl;

   return 0;
}


Try to follow the code to understand what the program does. The cout statement at the end is used to output the value of z. This value should appear in the output window near the bottom of the screen.

Lab exercise and demo

Grading scheme for each COMP2011 lab

  • Complete and demonstrate your solution to a TA, and answer a simple question in your lab session (1 mark)


Lab exercise: Compute the perimeter of a rectangle given the width and the height

Ask the user to input two numbers ( integers), namely, width and height, which represent the width and height of a rectangle. Calculate its perimenter and print the result to the screen. Note that you have to check whether width and height are positive before computing the result. If the width or height is zero or negative, print an error message to tell the user that the input has error and stop the program. A solution program (machine code runnable in Windows) can be downloaded: here for you to try out.

Sample #1:

Please input the width of the rectangle: 3
Please input the height of the rectangle: 4
The perimeter of the rectangle is 14

Sample #2:

Please input the width of the rectangle: 0
Please input the height of the rectangle: 5  
Error: the width and height must be positive!

Sample #3:

Please input the width of the rectangle: 5
Please input the height of the rectangle: -5 
Error: the width and height must be positive!


Once you finish the above exercise, you may find a TA to show the result.
